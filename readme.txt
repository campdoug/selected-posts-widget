== Description ==

Show selected posts using a widget.

== Installation ==

= Manual Installation =

1. Upload the entire `/selected-posts-widget` directory to the `/wp-content/plugins/` directory.
2. Activate Selected Posts Widget through the 'Plugins' menu in WordPress.
